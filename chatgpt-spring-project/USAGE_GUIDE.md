# 사업계획서 분석 시스템 사용 가이드

## 🎯 기능 소개

사업계획서(PDF, DOCX)를 업로드하면 ChatGPT가 자동으로 분석하여:
- **5개의 핵심 질문** 생성
- 각 질문당 **5개의 선택지** 제공
- 실시간 로딩 스피너로 진행 상황 표시

## 🚀 실행 방법

### 1. OpenAI API 키 설정
```bash
export OPENAI_API_KEY=your-api-key-here
```

### 2. 프로젝트 실행
```bash
./gradlew bootRun
```

### 3. 브라우저에서 접속
```
http://localhost:8080
```

## 📖 사용 방법

### Step 1: 파일 업로드
- 파일을 드래그 앤 드롭하거나 클릭하여 업로드
- 지원 형식: **PDF, DOCX**
- 최대 파일 크기: 10MB

### Step 2: 문서 내용 확인
- 업로드한 파일의 텍스트가 자동으로 표시됩니다
- 내용을 확인한 후 "질문 생성하기" 버튼 클릭

### Step 3: 로딩 대기
- 회전하는 스피너와 함께 "질문을 작성중입니다..." 메시지 표시
- ChatGPT가 문서를 분석하는 중입니다 (약 10-30초 소요)

### Step 4: 질문 및 선택지 확인
- 5개의 질문이 카테고리와 함께 표시됩니다
- 각 질문당 5개의 선택지(A~E) 중 하나를 선택할 수 있습니다

## 🎯 생성되는 질문 예시

**질문 1: 귀사의 주요 타겟 시장은 무엇인가요?** [시장분석]
- A. 대기업 (직원 1000명 이상)
- B. 중견기업 (직원 300-1000명)
- C. 중소기업 (직원 100명 이하)
- D. 스타트업 (직원 50명 이하)
- E. 개인 사업자

**질문 2: 예상 손익분기점은 언제인가요?** [재무계획]
- A. 1년차
- B. 2년차
- C. 3년차
- D. 4년차 이상
- E. 미정

## 🔧 커스터마이징

### 질문 개수 변경
`BusinessPlanAnalyzer.java`의 프롬프트에서 질문 개수를 수정할 수 있습니다:
```java
// 5개 → 10개로 변경
"중요: 정확히 10개의 질문을 생성하세요"
```

### 선택지 개수 변경
프롬프트에서 선택지 개수도 변경 가능합니다:
```java
// 5개 → 4개로 변경 (A, B, C, D만)
```

### 텍스트 길이 제한
`BusinessPlanAnalyzer.java`의 `limitTextLength` 메서드:
```java
// 10000자 → 20000자로 변경
String limitedText = businessPlanAnalyzer.limitTextLength(extractedText, 20000);
```
